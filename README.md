## Setup project
Frontend\
`cd frontend` then `yarn install` or `npm install`\
Backend\
`cd backend` then `yarn install` or `npm install`

## Run app
### frontend
Run 
`yarn start`

### backend
Run
`yarn server`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.\\
